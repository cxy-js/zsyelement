## 安装
```bash

npm install zsyelement
# or
yarn add zsyelement

```
### 全局导入
```bash
import zsyelement from "zsyelement"
import "zsyelement/dist/style.css"
app.use(zsyelement)
```
### 按需导入
```bash
import { zsyTabs } from "zsyelement"
import "zsyelement/dist/style.css"
```
#### 已有组件
+ zsyTabs
+ zsyButton
+ zsyInput
